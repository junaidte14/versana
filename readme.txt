== Description ==

Versana is a lightweight, versatile full site editing (FSE) block theme designed specifically for bloggers and content creators who value performance, flexibility, and professional design.

= Key Features =

* **Full Site Editing (FSE)**: Complete control over your site's appearance using the WordPress Site Editor
* **Two Header Layouts**: Pre-designed template parts with flexible layouts
* **Multiple Footer Variations**: 3 pre-designed footer templates to choose from
* **Performance Optimized**: Minimal custom CSS and performance enhancements
* **Translation Ready**: Fully prepared for multilingual sites

= Perfect For =

* Personal blogs
* Professional bloggers
* Content creators
* Magazine-style websites
* News sites
* Documentation sites

= Design Philosophy =

Versana follows a "theme.json first" approach, relying primarily on WordPress core functionality and theme.json for styling. This ensures:

* Faster page load times
* Better compatibility with future WordPress updates
* Easier customization through the Site Editor
* Cleaner, more maintainable code

= Customization =

All design customization (colors, typography, spacing, layouts) is done through the WordPress Site Editor:

1. Go to **Appearance → Editor**
2. Customize colors, fonts, and spacing using theme presets
3. Edit templates and template parts
4. Create and manage block patterns

Advanced customization can be added through a companion plugin if needed.

== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard
2. Navigate to **Appearance → Themes**
3. Click **Add New**
4. Search for "Versana"
5. Click **Install** then **Activate**

= Manual Installation =

1. Download the theme ZIP file
2. Log in to your WordPress dashboard
3. Navigate to **Appearance → Themes**
4. Click **Add New → Upload Theme**
5. Choose the ZIP file and click **Install Now**
6. Click **Activate**

= After Activation =

1. Go to **Appearance → Editor** to customize your site's design
2. Customize templates and template parts as needed
3. Start creating content!

== Frequently Asked Questions ==

= Is this theme free? =

Yes! Versana is 100% free and licensed under GPL v2 or later.

= Does this theme support WooCommerce? =

Not in v1.0.0. WooCommerce support is planned for a future release.

= Can I customize the colors and fonts? =

Absolutely! Use the Site Editor (**Appearance → Editor**) to customize all colors, typography, and spacing using the theme's preset values.

= How do I change the blog layout? =

You can customize blog layouts by editing templates in the Site Editor (**Appearance → Editor → Templates**). The theme includes pre-designed templates that you can customize or duplicate.

= Is the theme translation ready? =

Yes! Versana is fully translation ready. You can translate it using WordPress translation plugins or tools like Poedit.

= Does the theme support RTL languages? =

Yes, Versana includes RTL language support.

= How do I add my logo? =

1. Go to **Appearance → Editor**
2. Select the Header template part
3. Click on the Site Logo block
4. Upload your logo image

= Can I change the footer content? =

Yes! Go to **Appearance → Editor → Template Parts → Footer** to customize your footer content and layout.

= Where can I get support? =

For support questions, please visit the support forum on WordPress.org or contact us through our website at https://codoplex.com/contact
